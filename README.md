# Ejercicios DWES

Este repositorio contiene los ejercicios del módulo **Desarrollo en Entorno Servidor (DWES)**, enfocado en adquirir conocimientos fundamentales de **PHP**.  

![Portada DWES](./BootstrapTemplates/src/images/rel1.jpg)
---

## Superíndice de relaciones

En la raíz del proyecto se encuentra un **superíndice interactivo**, con enlaces que funcionan como tarjetas (`cards`) para acceder directamente a los índices de cada relación. Cada enlace incluye una breve descripción del contenido.  

| Relación | Descripción |
|----------|-------------|
| [Relación 1](./relacion1) | Variables, condicionales y bucles en PHP. Primeros 20 ejercicios con fundamentos básicos. |
| [Relación 2](./relacion2) | Formularios y validación. 20 ejercicios con GET, POST, validación en HTML/JS y estilado con Bootstrap. |
| [Relación 3](./relacion3) | Funciones y arrays avanzados. 19 ejercicios centrados en la creación de funciones y manipulación de arrays. |
| [Relación 4](./relacion4) | Sesiones, cookies, POO y JSON. 16 ejercicios sobre manejo de sesiones, POO y uso de JSON. |

> Cada tarjeta enlaza con el índice de su relación correspondiente, donde se puede acceder a los ejercicios individuales y sus descripciones.

---

## Bootstrap Templates

Contiene utilidades generales utilizadas en los ejercicios:  

- Plantillas de formularios con **Bootstrap**.  
- Objetos de validación en **JavaScript**: `Validator` y `ValidatorStyle`.  
- Imágenes útiles para índices y fondos de formularios.

---

## Nota importante

- Cada relación incluye un **índice interactivo** con tarjetas (`cards`) que representan las distintas actividades.  
- Al pasar el cursor sobre cada tarjeta, se muestra una breve descripción del objetivo del ejercicio.  
- El **superíndice en la raíz** permite acceder de manera rápida y directa a los índices de todas las relaciones, facilitando la navegación por el repositorio.
