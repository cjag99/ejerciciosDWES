# DWES Exercises

This repository contains the exercises for the **Server-Side Development (DWES)** module, focused on providing fundamental knowledge of **PHP**.  

![DWES Cover](./BootstrapTemplates/src/images/rel1.jpg)
---

## Relations Overview

At the root of the project, there is a **main interactive index** with links that act as cards to directly access the indexes of each relation. Each link includes a brief description of the content.  

| Relation | Description |
|----------|-------------|
| [Relation 1](./relacion1) | Variables, conditionals, and loops in PHP. The first 20 exercises covering basic concepts. |
| [Relation 2](./relacion2) | Forms and validation. 20 exercises using GET and POST methods, HTML/JS validation, and styling with Bootstrap. |
| [Relation 3](./relacion3) | Functions and advanced arrays. 19 exercises focused on creating functions and manipulating arrays. |
| [Relation 4](./relacion4) | Sessions, cookies, OOP, and JSON. 16 exercises on session handling, object-oriented programming in PHP, and using JSON. |

> Each card links to the corresponding relation index, where individual exercises and their descriptions can be accessed.

---

## Bootstrap Templates

Contains general utilities used in the exercises:  

- Form templates styled with **Bootstrap**.  
- Validation objects in **JavaScript**: `Validator` and `ValidatorStyle`.  
- Images useful for indexes and form backgrounds.

---

## Important Note

- Each relation includes an **interactive index** with cards representing the different activities.  
- Hovering over each card displays a brief description of the exercise objective.  
- The **main index at the root** allows quick and direct access to all relation indexes, making navigation through the repository easy.
